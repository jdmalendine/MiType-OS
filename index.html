Local Web Server Required

This application uses ES Modules and Import Maps, which modern browsers block when opening files directly from a folder (file://).

Option 1: Node.js (Easiest)

If you have Node.js installed, run this in your terminal:

npx serve .

Option 2: Python (Built-in on Mac/Linux)

Run this in your terminal:

python3 -m http.server

Option 3: GitHub Desktop / VS Code Extension

Use the Live Server extension in the Desktop version of VS Code (not the web version).
